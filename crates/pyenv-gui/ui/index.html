Understanding Shell Shims

When you run python or pip in your terminal, the shell calls the pyenv-native shims first. These shims intercept execution and dynamically route the call to the currently selected global, local, or virtual environment runtime. To verify active shim health manually at any time, execute pyenv doctor in your shell.

About Pyenv-Native

Pyenv-Native

Version …

A specialized, zero-dependency cross-platform engine that brings the pyenv workflow to Linux, macOS, and Windows — fast shims, managed virtual environments, and a unified GUI.

Written in pure, deeply typed Rust, pyenv-native bridges classic pyenv workflows with modern packaging, shell integration, and (on Windows) PEP-514 IDE discovery. Manage Python versions, virtual environments, and system configurations — all from one place.
